# bmi-calculator-python

  BMI Calculator / BMI Rechner

This is a Python-based tool developed for my IT-Ausbildung portfolio. It calculates the Body Mass Index and handles user input errors.

 Key Features / Hauptmerkmale
* **Error Handling:** Validates numeric inputs and prevents crashes.
* **Clean Code:** Structured with functions for reusability.
* **Bilingual:** Supports both English and German status outputs.

 Technologies
* **Language:** Python 3.x
* **Concepts:** Exception Handling, Math Operations, CLI.
